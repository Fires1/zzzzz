how to properly set up sharex

  1. download sharex, you can get it on steam: https://store.steampowered.com/app/400040
  2. run sharex after it downloads
  3. set up the default menu settings as you see fit, the only one i would recommend would be for it to auto-run whenever windows starts
  4. change your after capture tasks to upload
  5. set your destinations to be appropriate to your needs. by default it uses imgur for everything and its fine that way so you dont really need to change anything there
  6. open hotkey settings
  7. add one capture region task
  8. bind to your preferred screenshot key, i use keypad 7
  9. add gif screenrecording using custom region
    10.bind to appropriate key, i use kp_8

thats all needed, close the menu and make sure it appears in your taskbar
